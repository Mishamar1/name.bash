Мой первый пет-проект на GitHub

Программа для кодировки текста в азбуку морзе и ее раскодировки.

Для запуска программы выполните следующие действия:

Клонируйте репозиторий на свой ПК: 
  git clone https://github.com/Mishamar1/name.bash.git

Перейдите в директорию:
  cd name.bash

Далее запустите программу на языке С:
  gcc -o go morse.c && ./go

Для запуска программы на языке С++ введите:
  g++ -o go morse.cpp && ./go
